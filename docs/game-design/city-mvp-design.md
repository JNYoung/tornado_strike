# 城市场景 MVP 设计规格

更新时间：2026-06-09

## 关卡概念

玩家从城市中心的小龙卷风开始，通过吸收车辆和房屋逐步成长，最后吞噬发电厂、警察局、消防局等特殊建筑。第一版目标是验证成长曲线和场景密度，不追求最终美术品质。

## 场景组成

| 类型 | MVP 内容 | 后续扩展 |
| --- | --- | --- |
| 地面 | 无限流式城市区块、草地底板、城市街区、路缘 | 地块材质、对象池 |
| 道路 | 每区块十字道路、车道线、斑马线 | 弯道、桥梁、隧道 |
| 街道小物件 | 路灯、树、消防栓、长椅 | 垃圾桶、邮箱、广告牌 |
| 车辆 | 小汽车、公交车、车窗、车轮 | 卡车、警车、消防车、出租车 |
| 普通建筑 | 低矮房屋、门窗、屋顶水箱 | 商店、办公楼、公寓 |
| 特殊建筑 | 发电厂、警察局、消防局、标识轮廓 | 医院、学校、市政厅、电视塔 |
| 未来场景 | 雨林占位槽 | 树群、木屋、遗迹、瀑布 |
| 入口视觉 | 应用图标、开屏、主菜单 | Feature graphic、商店截图、动态菜单背景 |

## 吸收阶梯

| 阶梯 | 例子 | 建议 requiredRadius | 目标体验 |
| --- | --- | --- | --- |
| T1 | 小汽车、小道具 | 1.0 - 1.5 | 开局 10 秒内快速吸收 |
| T2 | 公交车、小房屋 | 1.8 - 2.4 | 第一次明显变大 |
| T3 | 大房屋、警察局、消防局 | 2.6 - 3.2 | 中期目标 |
| T4 | 发电厂、大型建筑 | 3.5 - 5.0 | 关卡高潮 |
| T5 | 地标、街区块 | 5.0+ | 后续版本 |

## 特殊槽位

特殊槽位不是单纯建筑，它们是系统扩展点。

### 发电厂

- 当前：可吸收特殊建筑。
- 后续玩法：吸收后获得短时间电光风暴，提升吸收半径。
- 变现槽位：奖励广告触发额外电力皮肤或金币翻倍。

### 警察局

- 当前：可吸收特殊建筑。
- 后续玩法：生成警车追逐或巡逻障碍。
- 任务槽位：吸收警察局前先吸收若干警车。

### 消防局

- 当前：可吸收特殊建筑。
- 后续玩法：消防车和水柱短暂减速。
- 任务槽位：完成消防局目标解锁火焰龙卷风皮肤碎片。

## 单局参数

| 参数 | 当前值 | 说明 |
| --- | --- | --- |
| 关卡时长 | 360 秒 | 目标 3-10 分钟，默认 6 分钟 |
| 目标分数 | 4200 | 当前无限城市 MVP 的完成目标 |
| 初始半径 | 1.25 | 可吸收小汽车 |
| 最大半径 | 8.0 | MVP 上限 |
| 地图边界 | 无限区块 | 玩家边界关闭，围绕龙卷风流式生成 |
| 区块大小 | 24 x 24 | 当前运行时生成单位 |
| 激活半径 | 2 区块 | 玩家周围常驻约 5 x 5 区块 |

## 无限场景规则

- `InfiniteCityWorld` 根据龙卷风当前位置生成周边区块，远离玩家的旧区块会自动销毁。
- 每个区块保持十字道路、街区、车辆和建筑密度，保证玩家移动时始终看到可吸收目标。
- 特殊建筑以低概率生成，并保留 `SceneSlot`，后续可以接任务、事件、广告奖励或关卡目标。
- 当前是 MVP 级流式生成，已经覆盖道路细节、车辆、房屋、街道小物件和特殊建筑；下一步需要接对象池，降低移动时频繁创建和销毁对象的开销。

## 反馈需求

当前已有：

- 吸收对象向龙卷风收缩。
- 龙卷风视觉尺寸变大。
- 分数和半径 HUD 更新。

下一步补：

- 吸不动对象显示红色轮廓或锁定提示。
- 吸收时加音效和短震动。
- 特殊建筑吸收时加屏幕震动和粒子爆发。
- 成长到新阶梯时显示短提示，例如 “公交车可吸收”。

## 美术方向

- 低多边形、明亮城市、清晰体量。
- 建筑高度不要太高，避免遮挡俯视视角。
- 小物件需要色彩区分，便于玩家判断目标。
- 特殊建筑用独特轮廓和颜色，第一眼可识别。

## 关卡验收

- 新玩家 5 秒内能移动。
- 新玩家 15 秒内能完成第一次成长。
- 新玩家 60 秒内能吸收房屋。
- 新玩家 360 秒内有机会完成目标。
- 屏幕上始终能看到下一个可吸收目标。
