# 龙卷突袭产品计划

更新时间：2026-06-09

## 产品定位

龙卷突袭是一款面向移动端的短局超休闲 3D 吸收成长游戏。玩家以龙卷风为主角，在城市、雨林等主题场景中吸收低等级物件，逐步成长到可以吞噬车辆、房屋和特殊建筑，并在限定时间内完成关卡目标。

## 第一阶段目标

优先跑通城市场景 MVP：

- 玩家可以通过拖动或 WASD 控制龙卷风移动。
- 龙卷风根据半径门槛吸收对象。
- 吸收后对象消失，分数增加，龙卷风半径和视觉尺寸增长。
- 城市场景包含小汽车、公交车、房屋。
- 发电厂、警察局、消防局作为特殊槽位先落地，后续接任务、事件、经济和美术替换。
- 支持 Android AAB 构建，Google Play 上架链路优先。
- UI 字符串具备 7 语言基础能力。

## 核心循环

1. 进入关卡，龙卷风初始半径较小。
2. 吸收路面小物件和小汽车，快速增长。
3. 解锁公交车、房屋等中型对象。
4. 成长到足够半径后吸收特殊建筑槽位。
5. 在倒计时内达到目标分数，完成关卡。
6. 结算后获得金币、皮肤材料或场景进度。

## 差异化方向

- 主题化槽位：城市中发电厂、警察局、消防局不是单纯障碍，而是关卡目标和系统扩展入口。
- 多生态：Android 优先，但从项目结构保留 iOS 和 Meta Quest 路径。
- 多语言首日内建：简中、繁中、英文、德文、法文、日文、阿拉伯文。
- 轻量灰盒先行：先验证吸收成长、地图节奏和 Android 上架链路，再投入高成本美术。

## 内容路线

### P0 城市 MVP

- 城市灰盒地图。
- 单局 120 秒。
- 物件体量阶梯：车辆 -> 房屋 -> 特殊建筑。
- 基础 HUD：标题、操作提示、分数、半径、时间、完成。
- Android Debug AAB 可构建。

### P1 城市可玩测试版

- 添加小道具：路灯、树、垃圾桶、邮箱、消防栓。
- 添加等级曲线和关卡目标配置。
- 添加低多边形视觉替换。
- 添加音效和震动。
- 添加简单新手引导。
- 接入 Firebase/Unity Analytics 或等价分析工具前，先完成隐私和数据安全决策。

### P2 Google Play 封闭测试版

- Release AAB 签名。
- 应用图标、Feature Graphic、手机截图、7 语言商店文案。
- Google Group 测试组。
- 封闭测试 12 人 14 天。
- 收集反馈并完成生产访问申请材料。

### P3 内容扩展

- 雨林场景：树木、巨石、木屋、遗迹槽位。
- 城市第二套建筑：商业区、高层楼、桥梁。
- 龙卷风皮肤和升级。
- 广告变现和内购前置设计。
